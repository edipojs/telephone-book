# Agenda telefônica (case técnico)

> Agenda telefônica com autenticação via código.

## Getting Started

### Prerequisites

Você precisará ter o **Docker** e o **Docker Compose** instalado.

### Installing

```
docker-compose up

```

## Built With

### Client
